## 1. 沖縄県庁の組織における食料関連の所管

### 1.1 農林水産部（食料政策の中核）

沖縄県庁農林水産部は、農業・林業・水産業に関する事項、農地関係の調整、土地改良、農林土木、漁港に関する事項を所掌する。食料関連では最も広範な所管を持つ部局である。

**本庁の課・室（12課）:**

| 課名 | 食料関連の主な所管 |
|------|-------------------|
| 農林水産総務課 | 部全体の総務・企画調整 |
| 流通・加工推進課 | 県産農林水産物の流通促進、6次産業化、地産地消推進、おきなわブランド確立 |
| 農政経済課 | 農業政策の企画立案、農業経営支援 |
| 営農支援課 | 農業者への営農指導・技術支援 |
| 園芸振興課 | 野菜・果樹・花卉の生産振興 |
| 糖業農産課 | さとうきび・穀類等の農産物振興 |
| 畜産課 | 畜産業の振興、食肉生産 |
| 村づくり計画課 | 農村地域の整備計画 |
| 農地農村整備課 | 農業基盤整備 |
| 森林管理課 | 森林資源の管理・活用 |
| 水産課 | 水産業振興、漁業管理 |
| 漁港漁場課 | 漁港整備、漁場管理 |

**研究機関:**
- 農業研究センター（本所+3支所）
- 畜産研究センター
- 森林資源研究センター
- 水産海洋技術センター（本所+石垣支所）
- 海洋深層水研究所

**流通関連:**
- 中央卸売市場（県産農林水産物の流通拠点）

**地域出先機関:**
- 北部農林水産振興センター
- 宮古農林水産振興センター
- 八重山農林水産振興センター

**その他:**
- 中部・南部農業改良普及センター
- 農業大学校
- 中央家畜保健衛生所、家畜衛生試験場、家畜改良センター
- 病害虫防除技術センター
- 栽培漁業センター

### 1.2 保健医療介護部（食品衛生・食品安全）

食品衛生法に基づく食品の安全確保、食品営業許可、食品衛生監視指導を所管する。

**食品関連の課・機関:**

| 課・機関名 | 食料関連の主な所管 |
|------------|-------------------|
| 薬務生活衛生課 | 食品衛生行政の総括、食品衛生監視指導計画の策定、食品表示 |
| 中央食肉衛生検査所 | 食肉の衛生検査（と畜検査） |
| 北部食肉衛生検査所 | 北部地域の食肉衛生検査 |
| 衛生環境研究所 | 食品の理化学検査、微生物検査 |
| 健康長寿課 | 県民の栄養改善、食生活指導、健康おきなわ21の推進 |

**保健所（食品衛生の実務執行）:**
- 北部保健所、中部保健所、南部保健所、宮古保健所、八重山保健所
- 各保健所の生活衛生班が食品関係営業許可、食品衛生監視指導を実施
- 那覇市は中核市のため、那覇市保健所が独自に所管

### 1.3 教育庁（学校給食・食育）

学校給食と学校における食育は、教育庁保健体育課が所管する。

| 課名 | 食料関連の主な所管 |

| 保健体育課 | 学校給食の実施・管理、食育の推進、栄養教諭の配置 |

**関連団体:**
- 公益財団法人沖縄県学校給食会（学校給食用食材の調達・供給）

> **注記:** こども未来部は子どもの貧困対策、こどもの居場所（こども食堂含む）を所管するが、学校給食の直接的な管轄は教育庁にある。

### 2.7 フードバンク・こども食堂に関する県の施策

**こども食堂（こどもの居場所）:**
- 沖縄県は「こどもの居場所（こども食堂等）実施状況調査」を毎年度実施
- 令和5年10月時点で県内316カ所、その後361カ所に増加
- 県内35市町村に展開
- 小学校区ごとの充足率（校区実施率）56.7%で**全国1位**
- 人口比箇所数も**全国1位**

**フードバンク:**
- NPO法人フードバンクセカンドハーベスト沖縄が2007年から活動
- 県内160カ所の連携施設・団体を通じて食品を無償提供
- 県は国の「フードバンク活動支援事業」の活用を推進
- フードドライブ実施店舗・窓口の情報提供

## 4. 食料安全保障の視点

### 4.1 台風時の物流途絶と食料供給への影響

#### 具体的事例: 2023年台風6号（カーヌン）

- **期間:** 2023年8月1日～（複数日にわたり暴風域）
- **影響:**
  - サンエー（県内最大手スーパー）: 本島全店舗で終日臨時休業、宮古島店舗も終日休業、石垣島店舗は午後1時閉店
  - イオン琉球: 全店臨時休業
  - ユニオン（24時間営業で知られる「最強スーパー」）: 停電により営業困難
  - コンビニ: 一部営業を継続するも、おにぎり・弁当・パン類がすぐに品切れ
- **台風通過後も:** 物流回復に時間がかかり、生鮮食品を中心に品薄状態が数日間継続

#### 具体的事例: 2024年台風3号

- 石垣市の大手スーパー各社が臨時休業
- 宮古島市の一部店舗も営業見合わせ

#### 構造的な脆弱性

- 暴風域に入るとスーパーはほぼ全店閉店
- 船舶・航空便の欠航により物流が完全に途絶
- **離島は二重の脆弱性**（本島→離島の二次輸送が途絶するため、回復がさらに遅れる）

### 4.2 県の食料備蓄体制

| 項目 | 内容 |
|------|------|
| 備蓄計画の根拠 | 沖縄県地域防災計画（令和7年3月修正が最新版） |
| 県の備蓄方針 | 発災から3日間の被災者の生命維持に最低限必要な物資を計画的に備蓄 |
| 県民への推奨 | 飲料水3日分（1人1日3L）、常食3日分（ご飯・ビスケット・乾パン等） |
| 大規模災害時 | **1週間分**の備蓄が望ましいとされている |

**課題:**
- 県レベルでの大規模な食料備蓄基地の整備に関する情報は確認できず
- 離島の市町村ごとの具体的な食料備蓄量・日数の公表データは限定的
- 台湾有事等の長期的な物流途絶を想定した備蓄計画は確認されていない

### 4.3 離島の食料備蓄体制

- 各離島市町村は地域防災計画に基づき備蓄を行っているが、具体的な備蓄量のデータは限定的
- 離島は台風による物流途絶が頻発するため、住民レベルでの備蓄意識は本島より高いとみられる
- ただし、長期的な物流途絶に対する制度的な備蓄体制は十分とは言えない

### 4.4 有事（台湾海峡有事等）における沖縄の食料供給リスク

#### 政府の避難計画（2025年3月公表）

- 武力攻撃予測事態を想定し、先島諸島5市町村から住民約11万人＋観光客約1万人＝計**12万人**を6日程度で九州・山口8県32市町に避難させる計画
- 民間フェリー・航空機を使用し、福岡空港・鹿児島空港・鹿児島港経由で避難先へ移動
- 受入期間: 約1カ月

#### 有識者の指摘

| 有識者・機関 | 主な指摘 |
|-------------|---------|
| キヤノングローバル戦略研究所 | 台湾有事でシーレーンが破壊されれば輸入が完全にストップ。輸入穀物に依存する日本の畜産はほぼ壊滅し、**シーレーン途絶から半年後には国民全員が餓死する可能性** |
| RIETI（経済産業研究所） | 「食料安全保障の研究：襲い来る食料途絶にどう備えるか」を2024年に刊行 |
| 防衛研究所 飯田将史部長 | 中国の軍事演習から台湾周辺シーレーン妨害の意図が明確。南シナ海に面した国々との物流は遮断される可能性 |
| PwC Japan | 日本に必要な食料安全保障政策を提言 |

#### 沖縄固有のリスク

- 沖縄は台湾から約600kmに位置し、**有事の際に最前線**となる
- 南西諸島のシーレーン（海上交通路）が妨害・封鎖された場合、沖縄への食料輸送は完全に途絶する可能性
- 食料自給率32%の沖縄は、物流途絶時に**約3分の2の食料が失われる**ことになる
- 避難計画は「予測事態」を前提としており、突発的な事態への対応は不透明

### 4.5 海上輸送に依存する食料サプライチェーンの脆弱性

```
[県外（本土・海外）]
  │ 海上輸送99% / 航空輸送1%
  ▼
[沖縄本島（那覇港等）]
  │ 本島内物流（トラック）
  ├──▶ 本島内のスーパー・市場
  │
  │ 二次海上輸送（フェリー等）
  ▼
[離島（宮古・八重山・久米島等）]
  │ 島内物流
  ▼
[離島のスーパー・商店]
```

**脆弱性のポイント:**
1. **単一モード依存:** 食料輸送の99%が海上輸送に依存
2. **二重のボトルネック:** 離島は本土→本島、本島→離島の2段階の海上輸送を経るため、どちらかが途絶すると食料が届かない
3. **代替手段の限界:** 航空輸送は容量・コスト面で海上輸送の代替にならない
4. **港湾集中リスク:** 那覇港への集中度が高く、港湾が使えなくなった場合のバックアップが限定的
5. **在庫の薄さ:** 島嶼県の特性として大規模な物流拠点・倉庫が限られ、在庫日数が短い

## 5. 沖縄の食料に関する固有課題

### 5.1 島嶼県特有の物流コスト構造

| 品目 | 沖縄の状況 |
|------|-----------|
| ガソリン（離島） | 本島比+20.2円/L、200円/L超（2023年） |
| 食料品全般 | 食料CPI 117.3 vs 全国平均115.2（2023年12月、那覇市） |
| 食品物価指数 | 106.4（全国平均を上回る） |
| 生鮮食品 | 関東比で5割以上高い場合も |

**コスト高の構造:**
- 県外からの海上輸送コスト（燃料費、船舶運航費）
- 離島への二次輸送コスト
- 小売規模の小ささによる規模の不経済
- 保冷設備・倉庫のコスト（亜熱帯気候のため冷蔵・冷凍コストが高い）

### 5.2 物価高騰の食料品への影響（2022年以降）

| 年 | 消費者物価上昇率（沖縄県） | 特記事項 |
|----|--------------------------|---------|
| 2022 | 高騰開始 | 食料指数で沖縄県が全国最高位に |
| 2023 | **3.6%** | **42年ぶり**の高い伸び幅 |
| 2024 | **3.2%** | 3%超が2年連続、10大費目全て増加 |

**影響の要因:**
1. 原油価格上昇による海上輸送コスト増大
2. 円安による輸入物価の高騰
3. 賃上げによる人件費増
4. **沖縄固有の要因:** 食料自給率が低いため、輸送費高騰の影響をより強く受ける

**県民生活への影響:**
- コロナ禍で既に疲弊した家計をさらに直撃
- 「買い控え」による消費低迷
- 特に低所得世帯・子育て世帯への影響が深刻

### 5.3 沖縄の伝統食文化（長寿食）の変容と現状

#### 伝統的な沖縄食文化の特徴

- **主なたんぱく源:** 豆腐・味噌など大豆由来の植物性たんぱく質が中心
- **豚肉文化:** 「鳴き声以外は全部食べる」と言われるほど豚の全部位を活用（ラフテー、ソーキ、中身汁等）
- **海藻・野菜:** もずく、アーサ、ゴーヤー、島野菜を豊富に使用
- **チャンプルー文化:** 豆腐・野菜・卵などを炒め合わせる合理的な調理法
- **植物性食品の割合が高い:** 食物繊維やポリフェノールの摂取量が多かった

#### 戦後の食文化変容

| 時期 | 変化 |
|------|------|
| 米軍統治時代（1945-1972） | コンビーフ・ポーク缶詰等の米軍軍用食料が大量流入。急速な食の欧米化 |
| 本土復帰後（1972～） | 本土のファストフード・外食チェーンの進出加速 |
| 1980年代～ | 若年層を中心に伝統食離れが進行 |
| 2000年～ | 「26ショック」発生。男性の平均寿命順位が急落 |

**「戦後版チャンプルー」の問題:**
- 琉球大学名誉教授・尚弘子氏が指摘
- 伝統的なチャンプルーに米軍由来のポーク缶等の加工肉が加わり、高脂肪化
- 行事食（クーブイリチー、中身汁等）は変化しにくかったが、日常食は大きく変容

### 5.4 ファストフード・外食産業の県内浸透度

| 指標 | 数値 |
|------|------|
| 県内ファストフード店舗数 | 334店舗 |
| A&W沖縄（日本唯一） | 30店舗 |
| 主なローカルチェーン | A&W、ジェフ、キングタコス等 |

**特徴:**
- **A&W（エンダー）** は1963年に日本初のファストフードレストランとして沖縄で開業。60年以上の歴史
- 米軍基地との歴史的関係から、アメリカンスタイルのファストフード文化が全国に先駆けて定着
- タコライス、ポーク卵おにぎりなど、米軍文化と沖縄食文化が融合した独自の食文化も発達
- 全国チェーン（マクドナルド、吉野家等）に加え、沖縄独自のローカルチェーンが多数存在

### 5.5 子ども食堂の数と活動状況

| 指標 | 数値 |
|------|------|
| こどもの居場所（こども食堂等）数 | **361カ所**（2024年9月時点） |
| 展開市町村 | 県内35市町村 |
| 小学校区充足率（校区実施率） | **56.7%（全国1位）** |
| 人口比箇所数 | **全国1位** |

**背景:**
- 沖縄県の子どもの貧困率は全国最悪水準（約29%）
- 子どもの貧困対策として県・市町村・民間が連携して「こどもの居場所」を積極的に整備
- 「沖縄こどもの未来県民会議」が支援団体の活動をサポート
- 全国1位の充足率は、課題の深刻さと同時に取り組みの積極性を反映

## 6. 他県・他国との比較

### 6.1 北海道（食料自給率200%超）との比較

| 項目 | 沖縄県 | 北海道 |
|------|--------|--------|
| カロリーベース食料自給率 | 32%（2021年） | 223%（2021年確定値）/ 213%（2023年概算値） |
| 全国順位 | 31位/47 | **1位**/47 |
| 食料供給の位置づけ | 食料の**消費地**（県外依存68%） | 日本最大の**食料供給地域** |
| 農地面積 | 限定的（島嶼地形） | 広大（日本の農地の約25%） |
| 主要農産物 | さとうきび、パイナップル、マンゴー、島野菜 | 米、小麦、じゃがいも、乳製品、ビート |
| 気候 | 亜熱帯海洋性 | 冷帯湿潤 |

**差異の主要因:**
- 農地面積の圧倒的な差
- 北海道は米・小麦・じゃがいもなどカロリーの高い主要穀物を大量生産
- 沖縄は土地が限られ、さとうきびなど特定作物に偏る構造

### 6.2 東京都（食料自給率0%）との比較

| 項目 | 沖縄県 | 東京都 |
|------|--------|--------|
| カロリーベース食料自給率 | 32% | **0%**（実数は約0.47%） |
| 生産額ベース食料自給率 | 約52% | 約2% |
| 全国順位 | 31位/47 | **47位**（最下位） |
| 人口 | 約147万人 | 約1,400万人 |
| 食料供給の特徴 | 海上輸送99%依存 | 陸上輸送で全国から調達可能 |

**比較のポイント:**
- 東京都は自給率が沖縄より遥かに低いが、**陸上輸送ネットワーク**により全国から食料を安定調達できる
- 沖縄は東京より自給率が高いものの、**海上輸送への依存**により物流途絶リスクが格段に高い
- 食料安全保障の観点では、自給率の数字だけでなく**輸送モードの冗長性**が重要

### 6.3 シンガポール「30by30」計画との比較

| 項目 | 沖縄県 | シンガポール |
|------|--------|-------------|
| 食料自給率 | 約32%（カロリーベース） | **約10%未満**（栄養必要量ベース） |
| 国土/面積 | 約2,281km2 | 約728km2（東京23区とほぼ同じ） |
| 食料安全保障政策 | 地産地消推進計画等 | **「30by30」ビジョン**（2030年までに栄養必要量の30%を国内生産） |
| 政策の位置づけ | 農業振興の一部 | **国家安全保障上の最優先課題** |

**シンガポール「30by30」計画の詳細:**
- 2019年発表。2030年までに栄養必要量の30%を現地生産で賄う目標
- シンガポール食品庁（SFA）が推進
- 農業生産性基金（APF）: 6,300万シンガポールドル（約49億円）を投入
- **ハイテク農業:** 室内農業（垂直農法）、細胞農業、代替プロテイン等に積極投資
- 2025年には省庁横断の「食料システムワーキンググループ（IFSWG）」設立
- **現状の自給率:** 葉物野菜14%、卵26%、魚10%

**沖縄への示唆:**
- シンガポールは沖縄より遥かに小さい国土面積で、食料自給率向上を**国家安全保障**として位置づけ
- ハイテク農業への大規模投資という戦略は、耕地面積が限られる沖縄にも参考になる
- 沖縄は食料安全保障を「防災」や「農業振興」の文脈で扱っているが、**独立した安全保障課題**として位置づける視点が不足

### 6.4 ハワイとの比較

| 項目 | 沖縄県 | ハワイ州 |
|------|--------|---------|
| 食料自給率 | 約32%（カロリーベース） | **約10～15%**（金額ベースで約20%） |
| 食料輸入依存度 | 約68%（県外） | **約90%**（州外） |
| 年間食料輸入額 | — | 約20億ドル（約3,000億円） |
| 食料安全保障政策 | 地産地消推進計画等 | **「Increased Food Security and Food Self-Sufficiency Strategy」** |
| 主要産業 | 観光・基地関連 | 観光・軍事基地 |
| 島嶼としての特性 | 太平洋の島嶼県 | 太平洋の島嶼州 |

**ハワイの食料安全保障戦略の詳細:**
- ハワイは食料の約90%を州外から輸入、年間食料輸入額は約20億ドル
- 一方で農業生産額の約80%を州外に輸出（パイナップル、マカダミアナッツ等の輸出向け作物に偏る構造）
- **3つの戦略目標:**
  1. 地元産食品への需要とアクセスの拡大
  2. 地元産食品の生産増加
  3. 食料自給ニーズを満たすための政策・組織支援
- **経済効果の試算:** 現在の食料輸入の10%を地元生産に置き換えるだけで、約3.13億ドルが州内に留まる
- 2025年にはAct 239に基づく「省庁横断食料システムワーキンググループ（IFSWG）」を設立

**沖縄との共通点:**
- 島嶼性による海上輸送依存
- 観光産業と軍事基地が主要産業
- 食料の大部分を域外から輸入
- 伝統的な農業の衰退と輸出向け作物への偏り

**沖縄への示唆:**
- ハワイは「輸入の10%を地元生産に置換」という具体的かつ現実的な目標を設定
- 省庁横断の食料システムワーキンググループという組織的アプローチは沖縄にも参考になる
- 沖縄もハワイも「観光と基地の経済」から「食料自給」への転換が共通課題

## 7. 総括と課題の構造

### 沖縄県の食料を巡る課題の全体構造

```
[構造的要因]
  島嶼県（海に囲まれた地理的条件）
  │
  ├─ 海上輸送99%依存（物流の脆弱性）
  ├─ 限られた農地面積（食料増産の限界）
  ├─ 米軍基地による土地利用制約
  └─ 亜熱帯気候（保冷コスト増、台風リスク）

[現状の課題]
  ├─ 食料自給率32%（県外依存68%）→ 年度変動大、安定しない
  ├─ 食料品価格の高さ（全国平均超、離島はさらに割高）
  ├─ 伝統食文化の変容 → 食の欧米化 → 平均寿命の低下
  ├─ 学校給食の地産地消率の低下（29%）
  ├─ 食品製造業の零細性（資金・原材料調達の課題）
  └─ 子どもの貧困（全国最悪水準）→ こども食堂全国1位の充足率

[リスク]
  ├─ 台風による短期的物流途絶（年に複数回発生）
  ├─ 台湾有事によるシーレーン封鎖（長期的物流途絶の可能性）
  └─ 物価高騰の継続（2022年以降3年連続の3%超上昇）

[行政対応の現状]
  ├─ 複数の計画策定（農林水産業振興計画、食育推進計画、地産地消推進計画、食品ロス削減計画等）
  ├─ 離島への物流補助（石油製品輸送補助等）
  └─ こどもの居場所（こども食堂）の積極的整備

[不足している視点]
  ├─ 食料安全保障を独立した政策課題として位置づける視点
  ├─ 台湾有事等を想定した長期的食料備蓄・自給戦略
  ├─ ハイテク農業（垂直農法、細胞農業等）への戦略的投資
  ├─ 省庁横断の食料安全保障専門組織
  └─ 離島ごとの食料供給リダンダンシー（冗長性）の確保
```
